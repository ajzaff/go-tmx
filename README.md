# go-tmx

A Go library for reading [Tiled maps](https://www.mapeditor.org/).

## Features

- Tile Animations
- Tile Objects
- WangSets
- Improved API

## Discontinued Features

- Decoding of uncompressed tile layers.
- Decoding of XML, and CSV tile layers.
